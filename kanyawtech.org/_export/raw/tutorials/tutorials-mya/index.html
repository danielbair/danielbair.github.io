သင့္ကြန္ပ်ဴတာျဖင့္ကရင္ဘာသာစကားသံုရန္ ဤ Unicode ကို install လုပ္ရန္လိုအပ္ပါသည္။ Unicode ဆိုသည္မွာကမၻာဘာစကားအားလံုးအတြက္အသံုးျပဳႏုိင္ေသာစနစ္ျဖစ္သည္။ Unicode version (၅.၀) တြင္ ကရင္ဘာသာစကားမ်ားစြာႏွင့္ျမန္မာဘာသာစနစ္မ်ားပါ၀င္သည္။ မ်က္ေမွာက္ကာလတြင္ ျမန္မာလူမ်ိဳးမ်ားသည္ ႏုိင္ငံတကာယူနီကုတ္စနစ္မဟုတ္ေသာ font မ်ားကိုသာသံုးစြဲေနျကေလသည္။ ဥပမာအားျဖင့္ Zawgyi၊ Ayar တို႕ျဖစ္သည္။ ဤ font
