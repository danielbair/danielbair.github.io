Time Machine - Troubleshooting

C11. Drive does not appear to be the correct backup volume for this computer (Cookies do not match)

Previous        Frequently Asked Questions        Home     Troubleshooting      Contact      Next 

 

This appears to occur only on Leopard.


If this happens after getting a new Logic Board, see item #C8.


This also happens on occasion after switching a Time Machine drive from one Mac to another, erasing your Time Machine disk/partition, or connecting a new Time Machine drive with the same name as an old one.


You can usually fix this by simply re-selecting your Time Machine drive in Time Machine Preferences (select the "none" drive, then the correct one).


If that doesn't help, try a full reset.  See item #A4.
